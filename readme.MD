# Introduction
提供了节假日相关api，包含：

+ 查询两个日期间的工作日天数
+ 根据日期和工作日天数计算截止日
+ 查询某个日期是否休假日
+ 查询某个月的休假日/调班情况

# Install
```
npm install
node run index.js
```
